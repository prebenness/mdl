### 3.C — Recognition accuracy metric

**Files:** `src/mdl/evaluation.py`, `tests/test_evaluation.py`

Three new functions:
- `compute_full_string_accuracy(forward_fn, inputs, targets, batch_size)` — per-string boolean (argmax correct at every position), batched with padding.
- `generate_negative_anbn(num_examples, max_n, p, seed)` — four types: wrong counts (m!=n), interleaved (abab), single-symbol, random.
- `compute_recognition_accuracy(forward_fn, pos_inputs, pos_targets, neg_inputs, neg_targets)` — returns accuracy, tp_rate, tn_rate.

Key finding: golden LM gets 0% recognition accuracy on valid strings because P(a)=0.7 > P(b)=0.3 at the a->b transition (memoryless geometric distribution). This is correct behavior — the metric distinguishes trained models that develop sharp counting from baselines.

### 3.A — Staged temperature annealing

**Files:** `src/mdl/training.py`, `differentiable_mdl.py`, `tests/test_staged_annealing.py`

Training module:
- Extended `MDLTrainState` with `freeze_mask: jnp.ndarray` (n_params,) float32.
- `create_mdl_state` initializes `freeze_mask=jnp.zeros(n_params)`.
- `compute_confidence_mask(logits, threshold)` — identifies peaked parameters (max softmax > threshold).
- `freeze_confident_weights(state, threshold)` — freezes peaked weights (one-hot logits x 100), unions with existing mask.
- `apply_freeze_mask_to_grads(grads, freeze_mask)` — zeros gradients for frozen parameters.
- `make_train_step` gains `use_freeze_mask: bool = False` parameter.

Orchestrator:
- CLI args: `--anneal_mode` (exponential/staged), `--staged_tau_levels`, `--staged_epoch_fracs`, `--freeze_threshold`.
- Staged branch in `run_training_basic`: validates config, iterates stages with constant tau, freezes between stages, resets optimizer.

Tests (10 tests in `tests/test_staged_annealing.py`):
- `TestFreezeMaskInit::test_freeze_mask_init_zeros`
- `TestConfidenceMask` (3 tests): uniform, peaked, threshold boundary
- `TestFreezeConfidentWeights` (2 tests): roundtrip one-hot, mask union persistence
- `TestGradientMasking` (3 tests): frozen zeroed, no-mask passthrough, other keys preserved
- `TestStagedTrainingSmoke::test_staged_training_3_stages` — 3 stages x 5 epochs

Smoke test verified:
